Chalice Text RPG DEMO Version 1.0 12/18/2020

PROGRAM DESCRIPTION
--------------------
This program is a text-only implementation of a simplified RNG-based role-playing game system developed with Java, Swing, and utilizing a bespoke soundtrack created by the developer.


GENERAL USAGE NOTES
--------------------
- This has not been tested as an executable, it is use via console purely for demonstration purposes.

- All controls are via the GUI. Console debug commands are to be implemented in a later version.
